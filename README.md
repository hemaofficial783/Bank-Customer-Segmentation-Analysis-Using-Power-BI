## Objectives
- Segment customers into Loyal, New, and Lost categories
- Analyze customer behavior using KPIs
- Identify high-value and low-value customer groups
- Support business decisions with interactive dashboards

## Tools & Technologies
- Power BI
- DAX
- Microsoft Excel / CSV
- Data Cleaning & Transformation

## Key Metrics
- Total Customers
- Total Revenue
- Transaction Frequency
- Average Revenue per Customer
- Customer Segments (Loyal, New, Lost)

## Dashboard Features
- Interactive slicers (Age, Gender, Segment)
- KPI cards for quick insights
- Segment-wise revenue analysis
- Trend analysis over time

## Key Insights
- Loyal customers generate the highest revenue
- New customers show strong growth potential
- Lost customers indicate areas for retention improvement
- Certain age groups contribute more to revenue

## Conclusion
Customer segmentation helps banks focus on high-value customers, improve retention strategies, 
and optimize marketing efforts using data-driven insights.
